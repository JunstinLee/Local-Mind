# Router 路由配置

本目录包含应用的路由配置模块，定义了应用的页面导航和路由规则。

## 文件结构

```
fronted/src/router/
└── index.js                 # 路由配置文件
```

## 路由功能说明

### index.js
- **功能**：应用的主路由配置文件
- **特性**：
  - 使用 Vue Router 4 实现路由管理
  - 配置了三种主要视图路由：传统文档管理、搜索功能和问答系统
  - 实现懒加载，提升应用性能
  - 设置根路径重定向到传统文档管理页面

#### 路由配置详情：
- `/` - 根路径，重定向到 `/traditional`
- `/traditional` - 传统文档管理页面
- `/search` - 搜索功能页面
- `/qa` - 问答系统页面

## 技术栈

- Vue Router 4 - Vue.js 官方路由管理器
- Vue 3 - 组合式 API 模式

## 设计原则

1. **懒加载**：使用动态导入实现组件懒加载
2. **模块化**：路由配置集中管理
3. **可扩展性**：易于添加新的路由规则
4. **用户友好**：设置默认重定向路径

## 使用方式

路由已在 main.js 中集成，组件可通过以下方式使用：

```javascript
// 在组件中
import { useRouter, useRoute } from 'vue-router'

// 或通过导航链接
<router-link to="/search">搜索</router-link>
```
